# ACE
ACE is a fall detection wearable based on Arduino Nano 33 IoT 
Has been developed as part of the Element14's challenge "Design for a cause"

## Projects

### Powersave
Sample application to put the SAMD21 in standby mode for a very short time (milliseconds) to save energy

### DataSetConverter
.NET console application to convert Sustemic dataset into timeseries suitable to be fed to the neural network I developed with TensorFlow

### Twilio
Test application for the Twilio services
